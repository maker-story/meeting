# 🎥 MultiMeet

A modern real-time video conferencing platform inspired by Google Meet, built for seamless online collaboration. MultiMeet enables users to create and join secure meeting rooms, communicate through high-quality audio and video, and collaborate from anywhere.

## ✨ Features

- 🔐 Secure user authentication
- 📹 Real-time video and audio communication
- 👥 Multi-user meeting rooms
- 🔗 Shareable meeting links
- 💬 Live chat during meetings
- 🎤 Microphone mute/unmute controls
- 📷 Camera on/off controls
- 🖥️ Responsive and user-friendly interface
- ⚡ Fast and lightweight performance

## 🛠️ Technologies Used

- Python
- fastapi
- WebRTC
- WebSockets
- HTML5, CSS3, JavaScript
- Tailwind CSS

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/multimeet.git
cd multimeet
```

### 2. Create a virtual environment

```bash
python3 -m venv venv
```

Activate it:

**macOS / Linux**

```bash
source venv/bin/activate
```

**Windows**

```bash
venv\Scripts\activate
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the project

```bash
python manage.py runserver
```

or (for Flask):

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:8000/
```
